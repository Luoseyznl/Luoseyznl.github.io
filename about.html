🧑‍💻 项目经历

ChatroomServer 聊天服务器 2025.02 -- 2025.07
C++, epoll, Kafka, Nginx, SQLite, RESTful API / 个人项目

简介：基于 C++ 实现的多线程高并发多人聊天服务器，设计并对比两种异步 IO 架构：

  • 架构一：采用线程池 + 路由分发模型，支持 RESTful API 接口，由 Nginx 实现反向代理与负载均衡。
  • 架构二：基于 epoll + 单 Reactor 异步事件驱动模型改进架构一，避免多线程锁竞争和切换开销。
  • 集成 Kafka 异步记录用户行为日志；封装线程安全的 DatabaseManager 操作本地 SQLite 存储模块。
  • 使用 wrk 工具进行高并发压测，结果表明第二种架构更加稳定，比如最大响应延迟降低约 90%。
